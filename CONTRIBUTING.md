# Contributing

Thanks for contributing to 100x100 Pomodoro.

## Setup
```bash
npm install
npm run dev
```

## Before opening a PR
```bash
npm run typecheck
npm run test
npm run build
```

## Pull Request Expectations
- Keep PRs focused and small when possible.
- Include a short description of behavior changes.
- Add or update tests for logic changes.
- Keep UI changes responsive for mobile and desktop.
